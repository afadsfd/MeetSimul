# 会议同传 - 版本更新说明

---

## v2.0.7 — 2026.03.22

### 界面优化

- 将「云端/本地模式切换」「词库」提升至顶部控制栏，与「边说边译」同级展示，不再隐藏在设置页
- 应用显示名称由 MeetSimul 改为「会议同传」
- 设置页新增「联系开发者」区块，展示开发者、邮箱、Telegram 联系方式
- 设置页底部新增版本号显示

---

## v2.0.6 — 2026.03.22

### 语音输入 & TTS 队列

- 新增麦克风语音输入功能，基于 macOS 原生 SFSpeechRecognizer 实现中文语音识别
- 支持连续说话：说完一句自动翻译、自动清空输入框、继续识别下一句
- 静默检测（0.8 秒）自动断句
- TTS 播放队列：连续多句翻译按顺序播放，不再互相打断
- 修复本地模式翻译不可用的问题（fallback 至云端翻译）

### 打包优化

- 生成带 Applications 快捷方式的标准安装 DMG
- 清理 4.3GB 构建缓存，项目体积从 4.5GB 降至 196MB

---

## v2.0.0 — 2026.03.21

### 全面重写

- **UI 重写**：深色主题改为白色 Apple 风格，全新设计语言
- **后端重写**：Python 后端完全移除，改用 Rust 原生实现
- **翻译引擎**：Google Translate + Youdao 双引擎云端翻译，LRU 缓存加速
- **TTS 引擎**：Edge TTS 语音合成（男声 Guy / 女声 Jenny）
- **本地模式**：支持下载离线模型（语音识别、翻译、语音合成）
- **术语管理**：自定义词库功能，支持添加专业词汇和人名
- **新增组件**：设置页、词库编辑器、语音选择器、引导页
- **BlackHole 检测**：自动检测虚拟音频驱动，提供一键安装

---

## v1.0.0 — 2026.03.15

### 初始版本

- 基于 Tauri + React + Python 后端架构
- 支持中文输入、云端翻译、英文语音播放
- 深色主题 UI
- BlackHole 虚拟音频支持
- 基础的会议同声传译功能
